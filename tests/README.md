# Test Suite

**See `TESTING_GUIDE.md` for comprehensive testing documentation.**

This directory contains the complete test suite for the residue scanning pipeline, including:
- 4 comprehensive tests with real PyRosetta computations
- 3 test PDB files (4ZFO, 5JXE, 7S7I)
- Reference CSV files for interface validation
- Custom mutation input CSV
- Test output directory (generated by running tests)

## Quick Start

```bash
# Run all tests
pytest tests/ -v -m slow -s

# Run specific test
pytest tests/test_comprehensive.py::test_interface_selection_4zfo_5jxe -v -m slow -s
```
